Quick Planner
Design System

A living reference for the design tokens, components, and patterns that power Quick Planner. Built for designers and engineers to explore, test, and grab production-ready code.

Explore the system

Foundations

Color ramps, semantic tokens, typography scale, spacing system, shadows, and radii. The building blocks of every component.

Atoms

Buttons, inputs, badges, status chips, and avatars. The smallest indivisible UI elements with interactive previews and copyable code.

Molecules

Kanban cards, toast notifications, dependency chips, and unscheduled task pills. Composed from atoms with clear data contracts.

Organisms

Kanban columns, modal dialogs, sidebar navigation, and header bar. Complex compositions that form distinct UI regions.

Templates

Full-page layouts showing how organisms compose into complete views. Live embed of the real application for context.

Patterns

Interaction patterns, animations, and accessibility guidelines. Coming soon.

Principles

Glassmorphism

Layered translucency with backdrop blur creates depth and hierarchy without heavy shadows.

Token-driven

Every color, radius, and shadow flows from design tokens. Change one value, update everywhere.

Theme-aware

Light and dark modes share semantic tokens. Components adapt automatically via CSS custom properties.

Accessible

WCAG 2.1 AA contrast ratios. Keyboard navigation. Reduced-motion support.
